The AgrospAI Portal uses environment variables to configure various aspects of the application. You can set these variables in a `.env` file at the root of your project.

## Getting Started

To set up your environment variables:

1. Copy the example file:
   ```bash
   cp .env.example .env
   ```

2. Edit `.env` with your desired values
3. Restart the application to apply changes

## Essential Configuration

These variables are required for basic functionality.

### Infura & WalletConnect

<ParamField path="NEXT_PUBLIC_INFURA_PROJECT_ID" type="string" default="xxx">
  Your Infura project ID for connecting to Ethereum networks. Get one at [infura.io](https://infura.io).
</ParamField>

<ParamField path="NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID" type="string" default="">
  Your WalletConnect project ID for wallet integration. Register at [walletconnect.com](https://walletconnect.com).
</ParamField>

### Market Fees

<ParamField path="NEXT_PUBLIC_MARKET_FEE_ADDRESS" type="string" default="0x9984b2453eC7D99a73A5B3a46Da81f197B753C8d">
  The ETH address where marketplace fees are sent.
</ParamField>

<ParamField path="NEXT_PUBLIC_PUBLISHER_MARKET_ORDER_FEE" type="string" default="0">
  Absolute fee taken by the publisher market upon ordering an asset. Declared on ERC20 creation.
</ParamField>

<ParamField path="NEXT_PUBLIC_PUBLISHER_MARKET_FIXED_SWAP_FEE" type="string" default="0">
  Percentage fee received by the publisher market when a datatoken is bought from a fixed rate exchange.
</ParamField>

<ParamField path="NEXT_PUBLIC_CONSUME_MARKET_ORDER_FEE" type="string" default="0">
  Absolute fee taken by the consume market upon ordering an asset.
</ParamField>

<ParamField path="NEXT_PUBLIC_CONSUME_MARKET_FIXED_SWAP_FEE" type="string" default="0">
  Percentage fee received by the consume market when a datatoken is bought from a fixed rate exchange.
</ParamField>

## Pricing Options

<ParamField path="NEXT_PUBLIC_ALLOW_FIXED_PRICING" type="boolean" default="true">
  Enable or disable fixed pricing option for publishers during price creation.
</ParamField>

<ParamField path="NEXT_PUBLIC_ALLOW_DYNAMIC_PRICING" type="boolean" default="false">
  Enable or disable dynamic pricing option for publishers during price creation.
</ParamField>

<ParamField path="NEXT_PUBLIC_ALLOW_FREE_PRICING" type="boolean" default="true">
  Enable or disable free pricing option for publishers. When enabled, datatokens are allocated to the dispenser contract.
</ParamField>

## Privacy & Compliance

<ParamField path="NEXT_PUBLIC_PRIVACY_PREFERENCE_CENTER" type="boolean" default="false">
  Enable GDPR-compliant privacy preference center to manage cookies. Requires configuration in `content/gdpr.json`.
</ParamField>

## Example Configuration

Here's a minimal `.env` file for production:

```bash
# Essential
NEXT_PUBLIC_INFURA_PROJECT_ID="your_infura_project_id"
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID="your_walletconnect_project_id"

# Marketplace
NEXT_PUBLIC_MARKET_FEE_ADDRESS="0x..."
NEXT_PUBLIC_METADATACACHE_URI="https://aquarius.pontus-x.eu"

# Features
NEXT_PUBLIC_ALLOW_FIXED_PRICING="true"
NEXT_PUBLIC_ALLOW_FREE_PRICING="true"
NEXT_PUBLIC_PRIVACY_PREFERENCE_CENTER="true"

# External Services
AGROPORTAL_API_KEY="your_agroportal_api_key"

# Monitoring
NEXT_TELEMETRY_DISABLED=1
```

## Security Notes

<Warning>
  Never commit your `.env` file to version control. Always use `.env.example` as a template and keep sensitive values private.
</Warning>

<Info>
  Environment variables prefixed with `NEXT_PUBLIC_` are exposed to the browser. Never put sensitive secrets in these variables.
</Info>
