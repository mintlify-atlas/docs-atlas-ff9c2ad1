The `SearchPage` component provides a comprehensive search interface with filtering, sorting, and pagination capabilities.

## Component Signature

```tsx
export default function SearchPage({
  setTotalResults,
  setTotalPagesNumber
}: {
  setTotalResults: (totalResults: number) => void
  setTotalPagesNumber: (totalPagesNumber: number) => void
}): ReactElement
```

## Usage

```tsx
import SearchPage from '@components/Search'
import { useState } from 'react'

function Search() {
  const [totalResults, setTotalResults] = useState(0)
  const [totalPages, setTotalPages] = useState(0)
  
  return (
    <>
      <h1>Found {totalResults} assets</h1>
      <SearchPage
        setTotalResults={setTotalResults}
        setTotalPagesNumber={setTotalPages}
      />
    </>
  )
}
```

## Features

### URL-based Search

The component reads search parameters from the URL query string:

```tsx
const parsed = queryString.parse(location.search, {
  arrayFormat: 'separator'
})
```

### Debounced Search

Search requests are debounced to prevent excessive API calls:

```tsx
const fetchAssets = useDebouncedCallback(
  async (parsed: queryString.ParsedQuery<string>, chainIds: number[]) => {
    setLoading(true)
    const queryResult = await getResults(parsed, chainIds, newCancelToken())
    setQueryResult(queryResult)
    setTotalResults(queryResult?.totalResults || 0)
    setTotalPagesNumber(queryResult?.totalPages || 0)
    setLoading(false)
  },
  500
)
```

### Pagination

Automatic page updates with URL synchronization:

```tsx
const updatePage = (page: number) => {
  const newUrl = updateQueryStringParameter(
    pathname + '?' + queryString,
    'page',
    `${page}`
  )
  return router.push(newUrl)
}
```

## Layout

The component renders with a two-column layout:

```tsx
<div className={styles.container}>
  <div className={styles.filterContainer}>
    <Filter addFiltersToUrl expanded />
    <Sort expanded />
  </div>
  <div className={styles.results}>
    <AssetList
      assets={queryResult?.results}
      showPagination
      isLoading={loading}
      page={queryResult?.page}
      totalPages={queryResult?.totalPages}
      onPageChange={updatePage}
      showAssetViewSelector
    />
  </div>
</div>
```

## Query Parameters

Supported URL query parameters:

- `page` - Current page number
- `sort` - Sort field and direction
- `accessType` - Filter by access type (access/compute)
- `serviceType` - Filter by service type (dataset/algorithm/saas)
- `tags` - Filter by tags
- `chainIds` - Filter by blockchain networks
