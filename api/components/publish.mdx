The `PublishPage` component provides a complete publishing workflow for creating NFTs, datatokens, and metadata for data assets.

## Component Signature

```tsx
export default function PublishPage({
  content
}: {
  content: { title: string; description: string; warning: string }
}): ReactElement
```

## Publishing Workflow

The component orchestrates a 3-step publishing process:

### Step 1: Create NFT & Datatokens

```tsx
async function create(values: FormPublishData): Promise<{
  erc721Address: string
  datatokenAddress: string
}>
```

Creates the ERC721 NFT and datatoken contracts with pricing schema.

**Returns:**
- `erc721Address` - Address of the created NFT contract
- `datatokenAddress` - Address of the created datatoken contract

### Step 2: Encrypt DDO

```tsx
async function encrypt(
  values: FormPublishData,
  erc721Address: string,
  datatokenAddress: string
): Promise<{ ddo: DDO; ddoEncrypted: string }>
```

Constructs and encrypts the DDO (Decentralized Data Object) using the Provider service.

**Returns:**
- `ddo` - The constructed DDO object
- `ddoEncrypted` - Encrypted DDO string

### Step 3: Publish Metadata

```tsx
async function publish(
  values: FormPublishData,
  ddo: DDO,
  ddoEncrypted: string
): Promise<{ did: string }>
```

Writes the encrypted DDO into the NFT metadata.

**Returns:**
- `did` - The Decentralized Identifier for the published asset

## Features

- **Multi-step Form**: Guided workflow with validation at each step
- **Error Recovery**: Ability to retry failed steps without restarting
- **Purgatory Check**: Prevents publishing from accounts in purgatory
- **Automation Support**: Integrates with automation wallet for automated publishing
- **Network Validation**: Ensures correct network configuration
- **Debug Mode**: Optional debug panel for development
